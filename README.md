# MBTI Compatibility Tool

MBTI（16タイプ診断）の結果をもとに、自分と相手の相性を診断するツールです。

## 開発の目的と背景
日常のコミュニケーションにおいて、自分と相手の性格特性の違いを理解し、より円滑な関係構築を支援することを目的に作成しました。直感的な操作と、一目でわかる視覚的なデザインを意識しました。

## 使用技術
- React 19 (Vite)
- Tailwind CSS
- JavaScript (ES6+)

## セットアップ方法
```bash
# リポジトリをクローン
git clone [https://github.com/hasimoto0203/mbti.git](https://github.com/hasimoto0203/mbti.git)

# ディレクトリに移動
cd mbti

# パッケージをインストール
npm install

# ローカルサーバーを起動
npm run dev
